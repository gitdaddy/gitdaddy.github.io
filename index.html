3D Star Fight

This 3D star fight project was done with the help of THREE.js built off of a WebGL framework. In this project I learned much about how meshes are used in graphics to create objects. This type of a game requires no server or back-end.
Keys include WASD - pitch, yaw and roll, Q and E - forward and backward, space - fire. Press 2 for two player mode which uses IJKL for pitch, yaw and roll, U and O - foward and backward. - Takes a second to load
