Recent Projects

3D Asteroids with JavaScript Graphics

The 3D asteroids project was done with the help of THREE.js which is a WebGL framework. Keys include WASD for direction, IK for pitch and JL for the roll. Collision Detection Still in progress. Feel free to give it a try!

Event Wiki

The Event Wiki Web App allows users to create and manage their own events. The application was built using Java Servlets with JSP. This application uses mySQL database technology to store events with their corresponding images.

Lord of the Rings Survey

This project was done as an exercise in using HTML 5 LocalStorage, and JavaScript charts. This web page takes the user through an interactive brief survey about their favorite characters, weapons and places from the series. All survey results are stored using Local Storage.
